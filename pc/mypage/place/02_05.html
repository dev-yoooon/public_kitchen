가이드라인 운영원칙

가이드 확인 후 매력적인 공간을 등록해 주세요

운영원칙

  • 공간 공유 시 예상 수익 금액(4시간 기준)

    인원수 쿠킹/베이킹전문 스튜디오 요리시설
    6명 이하 이용시 80,000원 보유
    60,000원 미보유
    7명 이상 이용시 110,000원 보유
    90,000원 미보유

  • 취소보상금 제도

    공공의 주방에서는 선생님이 클래스를 취소할 경우 호스트님의 공간 공유를 보상해주는 취소보상금 제도를 제공하고 있습니다

  • 선생님 클래스 취소 시

    취소일 호스트 보상금
    5일전 공간대여 금액의 50%
    4일전 공간대여 금액의 100%
    3일전 공간대여 금액의 30%
    2일전 공간대여 금액의 30%
    1일전 공간대여 금액의 30%

  • 서비스 수수료

    공공의 주방에서 호스트로 활동 시 클래스 종료 후 정산 수입 금액 중 일부를 수수료로 지불하게 됩니다

공간소개

  • 공간 명

    공유하고자 하는 공간 명을 작성해 주세요
    예시 : OO 쿠킹 스튜디오

  • 공간 한 줄 설명

    공간명에서 적지 못했던 추가 설명을 적어보세요
    예시 : 최대 20명까지 수용가능한 최고의 요리시설

  • 해시태그

    많은 사람들이 검색해 볼 수 있게 태그를 입력해주세요
    예시 : #쿠킹스튜디오 #강남 #서울

  • 커버 이미지

    메인 썸네일은 홈페이지 메인이나 공간 목록에서 노출되는 이미지를 뜻합니다

  • 상세 이미지 등록

    상세 썸네일은 클래스 클릭 시 노출되는 첫번째 이미지를 뜻합니다
    상세 썸네일영역에 노출되며 최대 9개 까지 추가 등록이 가능합니다
    - 공간 내부를 잘 보여주는 사진을 사용해 주세요
    - 공간의 세부적인 특징이 잘 들어나도록 사진을 찍어주세요
