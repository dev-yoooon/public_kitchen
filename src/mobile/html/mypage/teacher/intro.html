선생님 신청하기

STEP 01 선생님 신청하기

  • ① 하단 선생님 신청하기 버튼 클릭
  • ② 기본/상세 정보를 입력
  • ③ 관리자 승인 대기

STEP 02 클래스 개설하기

  • ① 마이페이지>진행클래스>신규버튼 클릭
  • ② 기본/상세 정보를 입력
  • ③ 관리자 승인 대기

STEP 03 클래스 관리하기

  • ① 마이페이지>진행클래스 클릭
  • ② 수강생 모집현황 실시간 확인 가능
